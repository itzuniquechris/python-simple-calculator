# 🧮 Python Simple Calculator (Version 1)

A beginner-friendly command-line calculator built using Python.

This project performs basic arithmetic operations and includes input validation, formatted output, and divide-by-zero protection.

---

## 🚀 Features

* Addition
* Subtraction
* Multiplication
* Division
* Input validation
* Menu-based system
* Loop for continuous calculations
* Exit option
* Divide-by-zero protection

---

## 💻 Example Output

```text
--- Simple Calculator ---

1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)
5. Exit
```

---

## 🧠 Concepts Used

* while loop
* if / elif / else
* user input
* float conversion
* string validation with isdigit()
* arithmetic operators
* formatted output (.2f)
